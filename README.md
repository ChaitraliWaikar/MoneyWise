# MoneyWise

MoneyWise is a simple web app that helps users learn about budgeting and manage their finances better.  
It uses two agents â€” one for giving budgeting tips and another for answering basic financial questions using AI.

## Tech Stack

- Python (Flask)  
- HTML, CSS  
- Gemini AI (via Google Generative AI API)  

## How to Run

1. Clone the repo  
2. Install dependencies: `pip install -r requirements.txt`  
3. Add `.env` file with your API keys  
4. Run: `python app.py`  
5. Open `http://localhost:5000`
